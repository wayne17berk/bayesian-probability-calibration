# Bayesian Probability Calibration Framework

## Overview
Bayesian updating system to recalibrate fair value estimates with tail calibration diagnostics.

## Implemented Features
- ✅ Bayesian updating with weighted observations
- ✅ Tail calibration for extreme outcomes
- ✅ Brier score calculation for calibration 

## Setup
```bash
pip install -r requirements.txt
python calibrator.py
```

## Tech Stack
Python, NumPy
